Commands:-

1.shutdown

Typing this will just close the program.

2.write

Typing this will ask you for a location and then it will allow you to write into that file.

3.files

4.content

Typing this will allow you to see the contents of a file

5.dir [path]

Typing this(Replace path with a path) will show you all the files in a directory or a folder.

6.remove

Typing this will allow you to remove a file.

7.create

Typing this will allow you to create a file.

8.exists

Typing this will allow you to see if a file exists.

9.cd [path]

Typing this will allow you to open a file.

10.restart

Typing this will restart the program

11.rename

Typing this will rename a file.

12.change password

Typing this will change the password.

13.download

Typing this will let you download files from the web.

14.remove password

It removes the current password

15.set password

It sets a password.

16.history

It shows all operations performed on files with Samex.

17.clear history

clears the history.

18.secret files

h3>Lets you write and view encrypted files

19.emdir

wipes a directory>

20.showalpha

displays all the files of a folder alphabetically.
